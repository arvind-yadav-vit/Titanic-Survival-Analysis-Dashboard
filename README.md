# Titanic-Survival-Analysis-Dashboard

## Project Overview
This project analyzes Titanic passenger survival data using Tableau Public.

## Dashboard Insights
- Survival Count Analysis
- Gender-wise Survival Analysis
- Passenger Class Survival Analysis
- Age Distribution of Passengers
- Fare vs Survival Status Analysis

## Tools Used
- Tableau Public
- Data Visualization
- Data Analysis
